# Libreria Assembly CP/M-like

`retronet-asm` non supporta ancora `.include` o macro. Per ora questa cartella
contiene blocchi copiabili negli esempi `.COM`.

- `cpm-bdos.asm`: costanti BDOS e DMA usate dagli esempi.

Quando arrivera' il supporto include, questi file potranno diventare una piccola
libreria riusabile direttamente dal sorgente.
